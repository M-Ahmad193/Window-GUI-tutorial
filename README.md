## üöÄ Getting Started

### Prerequisites

To use the code examples from this tutorial, you'll need:

- **Windows Operating System** (Windows 7 or later)
- **C++ Compiler** (Visual Studio, MinGW, or similar)
- **Basic C++ Knowledge**

### Running the Tutorial Website

1. **Clone the repository**
   ```bash
   git clone https://github.com/M-Ahmad193/Window-GUI-tutorial.git
   cd windows-gui-programming
   ```

2. **Open the website**
   - Simply open `web.html` in your web browser
   - No build process or dependencies required!

3. **Start Learning**
   - Navigate through the tutorial sections
   - Try the interactive demos
   - Copy and test the code examples in your C++ IDE

### Hosting Online

You can easily host this tutorial website using:

- **GitHub Pages**: Push to GitHub and enable Pages in repository settings
- **Netlify**: Drag and drop the files to [Netlify Drop](https://app.netlify.com/drop)
- **Vercel**: Deploy with a single command using Vercel CLI
- **Any Static Hosting**: Upload the HTML files to any web server
